# Ảnh học viên — section Case Studies (trang chủ SBB)

Thả 3 file ảnh chân dung vào đúng thư mục này với **đúng tên** sau:

| File | Học viên | Card |
|------|----------|------|
| `my-linh-le.jpg`   | My Linh Le         | BUSINESS    |
| `khanh-giang.jpg`  | Khanh Trong Giang  | HOSPITALITY |
| `victoria-tran.jpg`| Victoria Tran      | MARKETING   |

## Yêu cầu ảnh
- **Hướng:** dọc (portrait), tỉ lệ ~3:4 hoặc 4:5.
- **Kích thước tối thiểu:** ~800 × 1000 px (càng nét càng tốt).
- **Bố cục:** khuôn mặt nằm ở **phần trên–giữa** khung (ảnh được crop `object-position: center 20%`).
- **Định dạng:** `.jpg` (hoặc đổi đuôi trong `index.astro` nếu dùng `.png` / `.webp`).

Sau khi thả file đúng tên, ảnh sẽ tự thay cho ô "Ảnh học viên — đang cập nhật".
Nếu cần chỉnh vùng crop cho từng ảnh, báo mình tinh chỉnh `object-position`.
